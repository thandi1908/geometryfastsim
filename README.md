# geometryfastsim

This project develops a generative model for fast calorimeter simulation that conditions on detector geometry, enabling generalisation across calorimeter configurations, material compositions, and particle types — including geometries unseen during training. The architecture combines Mamba state-space models with Graph Neural Networks to jointly process mesh-based geometry representations and point cloud shower outputs. Successfully demonstrating such generalisation would represent a significant advance over the current state of the art, with direct applicability to detector design studies for next-generation experiments where rapid simulation of candidate geometries is essential.

This repository will be populated with the full source code upon publication.
